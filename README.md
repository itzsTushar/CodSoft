# Machine Learning Internship Projects – CodSoft

This repository contains the projects I completed as part of my **Machine Learning internship at CodSoft**. The internship required completing multiple hands-on tasks to strengthen skills in data preprocessing, model building, evaluation, and deployment. Each task focuses on applying machine learning techniques to solve real-world problems.

## 🛠️ Tasks Overview

### **Task 1: Movie Genre Classification**
- **Objective:** Predict the genre of a movie based on its plot summary or textual information.  
- **Techniques:** TF-IDF, word embeddings, Naive Bayes, Logistic Regression, Support Vector Machines (SVM).  
- **Dataset:** [Click here](#)  

### **Task 2: Credit Card Fraud Detection**
- **Objective:** Detect fraudulent credit card transactions using transaction data.  
- **Techniques:** Logistic Regression, Decision Trees, Random Forests.  
- **Dataset:** [Click here](#)  

### **Task 3: Customer Churn Prediction**
- **Objective:** Predict customer churn for a subscription-based service or business using historical data.  
- **Techniques:** Logistic Regression, Random Forests, Gradient Boosting.  
- **Dataset:** [Click here](#)  

### **Task 4: Spam SMS Detection**
- **Objective:** Classify SMS messages as spam or legitimate.  
- **Techniques:** TF-IDF, word embeddings, Naive Bayes, Logistic Regression, Support Vector Machines (SVM).  
- **Dataset:** [Click here](#)  

### **Task 5: Handwritten Text Generation**
- **Objective:** Generate handwritten-like text using a character-level recurrent neural network (RNN).  
- **Dataset:** [Click here](#)
